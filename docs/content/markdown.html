Markdown

Markdown files

For simple pages (like documentation) you can choose to write an entire file in markdown.

The default markdown.template.html looks and expects that you have a base.template.html with a block named content.

{% code "html" %}{% raw %} {% extends "base.template.html" %} {% block content %} {% markdown %} {{ content }} {% endmarkdown %} {% endblock %} {% endraw %}{% endcode %}

You can easily overwrite how markdown files are rendered by putting your own copy of markdown.template.html at the top of your site.

frontmatter

Markdown template tags

Use the {% raw %}{% markdown %}{% endraw %} tag for a simpler writing experience, right in the middle of any HTML file.

{% code "html" %}{% raw %} {% extends "base.template.html" %} {% block content %}

Some HTML!

{% markdown %} ## And content {% endmarkdown %}

You can also include a markdown file

{% markdown %} {% include "README.md" %} {% endmarkdown %} {% endblock %} {% endraw %}{% endcode %} {% endblock %}
