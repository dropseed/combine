Combine

A straightfoward static site generator for building simple HTML websites, with the power of Jinja templates.

How it works

You've got a directory named content, and that's where your website goes.

{% code "bash" %} content ├── assets │   └── css │    └── style.css |   └── img │   └── logo.png ├── base.template.html ├── favicon.ico └── index.html └── about.html {% endcode %}

It doesn't really matter where you put things, so do what works for you. You can read about the specifics later.

When you run combine build or are developing with combine work, you'll get an output directory with something like this.

{% code "bash" %} output ├── assets │   └── css │    └── style.css | └── img │   └── logo.png ├── favicon.ico └── index.html └── about    └── index.html {% endcode %}

That's about it! You can leverage Jinja to build a templating system that works exactly how you want it to, and off you go.

Getting started

Markdown

{% code "html" %}{% raw %} {% markdown %} # A heading! {% endmarkdown %} {% endraw %}{% endcode %}

Code highlighting

Combine comes with pygments and a {% raw %}{% code %}{% endraw %} tag to easy add highlighted code sections to your html.

You can use it like this:

{% code "html" %}{% raw %} {% code "python" %} def test(): print("hi!") {% endcode %} {% endraw %}{% endcode %}

And you'll get this:

{% code "python" %} def test(): print("hi!") {% endcode %}

To style it, you just need to add some CSS. Run combine utils highlight-into to get a list of available themes and the customizable CSS. Put that CSS in your stylesheet somewhere and tweak it if you want.

Prefix the CSS with .highlight for more specificity:

{% code "css" %} .highlight .hll { background-color: #ffffcc } .highlight .c { color: #008800; font-style: italic } /* Comment */ .highlight .err { border: 1px solid #FF0000 } /* Error */ .highlight .k { color: #AA22FF; font-weight: bold } /* Keyword */ .highlight .o { color: #666666 } /* Operator */ {% endcode %}
